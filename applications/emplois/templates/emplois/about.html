About page

This website provides the current and past job opportunities at the City of Ottawa. The data obtained comes directly from the Ottawa Open data portal.

À Propos

Cette page web fournit la plus récente liste d'emplois à Ville d'Ottawa. Les emplois qui ne sont plus affichés sur le site web de la ville parce qu'ils ont été expirés, sont aussi accessible sur ce site. Les données ont été récupérés sur le portail de données ouvertes d'Ottawa

Le contenu de ce site web est sous license CC-BY-ND. L'image du drapeau de la ville d'Ottawa est sous license Wiki Creative Commons. Les scripts qui ont permis de créer cette application web sont aussi disponible sur Github sous license CC-BY. Les scripts contenants des informations personnelles par exemple pour faire fonctionner la publication de Tweet sur Twitter n'ont pas été inclus dans le code sur Github
